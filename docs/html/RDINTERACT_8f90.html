Detailed Description

Reads the file 'interaction'.

This file defines all interactions between particles based on their type. The file has several sections, in the following order.

First the type number, name, mass and whether it is an atom or bead is defined: 

 atom_types 6 
 # Type, Label, Mass,          A/B
    1	   H      0.00100782     A
    2	   Hn     0.00100782     A
    3	   C      0.0120         A
    4	   Ca     0.0120         A
    5	   N      0.0140031      A
    6	   O      0.0159949      A

Next bonds are defined. All bonds are order independent, ie a bond between types 1-3 is the same as a bond between types 3-1.

 bond_types 5
 #Type1, Type2, filename
 3	   1	  bond1
 3	   4	  bond3
 4	   6	  bond7
 4	   5	  bond4
 5	   2	  bond5
 angle_types 3
 #Type1, Type2, Type3, filename
  7      3      1      ang.201
  7      3      4      ang.202
  1      3      1      ang.002
 torsion_types 1
 #Type1 Type2 Type3 Type4 filename
 2      5     4     6	    tors5

Then nonbonded potentials are defined

non_bonded_interaction_types 3
#Type1 Type2     filename
 1     1         nb01
 1     2         nb02
 1     3         nb03

Finally out of planes torsions are defined. Note that all sections must be defined, even if they are empty, as shown here.

 Out_of_Planes 0
 #Type1 Type2 Type3 Type4 filename
