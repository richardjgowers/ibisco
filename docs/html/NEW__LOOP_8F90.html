subroutine NEW_LOOP ( )

The molecular dynamics loop.

Called from IBIsCO()

Calls a series of subroutines to perform the MD loop
SHIFT()
VIRTUAL_DEF() If using hybrid system
UPDATE_NEIGHBOURLIST() If required (updates every n steps)
NEW_FORCE() Main calculation step
MOMENTUM() If required
SCALEBP() If NPT
AVERAGE() Keeps track of average physical values for reporting
WRITETRJ() Writes the trajectory
OUTPUT() Writes the s-md.out file
TIMING() If compiled using TIMING flag, reports on time in each subroutine
