00005 !       SETS UP A LIST OF THE TWENTY SIX NEIGHBOURING CELLS OF EACH OF
00006 !                    THE SMALL CELLS IN THE CENTRAL BOX.
00007 
00008 ! There are 6 cells for each face of the cell, 12 cells for each edge and 8 cells for each vertex. 26 neighbour cells for each cell
00009 
00010 ! In the vector MAP is stored the position of the 13 neighbours cells for each cell, the counter (IMAP) advances in 13 of 13.
00011 ! e.g. For the cell 1
00012 ! ICELL(1,1,1) = 1
00013 ! IMAP = (ICELL ( IX, IY, IZ ) - 1 ) * 13 = 0
